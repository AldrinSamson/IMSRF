FREQUENTLY ASKED QUESTIONS (FAQs)

 

For all the Red Bank users.

 

 

How to add new user accounts?

 

Only the System Administrator has access to adding new accounts for all the users of the internal system. The System Administrator must get the basic information of the users, including the active email address and their profile photo, then create a default password for them.

 

 

STEP 1:  As the System Administrator, you must go to the Accounts Page and press the Add User button and fill up all the fields in the Add New User form.

 

 

How to change default password?

 

On the upper left of your screen, click Go to your Profile and select the Change Password button. Passwords must be at least 6 alphanumeric characters long.

 

 

How to manage blood requests?

 

Only the users with a role of Dispatch & Request Manager and System Administrator have access in managing the blood requests.

 

 

STEP 1: Go to the Requests Page and you’ll see five (5) tabs, which are: For Approval, Approved, Partner Request, Denied, and Archive.

 

STEP 2: To view and validate the pending new blood requests, you must go to the For Approval tab.

 

STEP 3: To view and create a dispatch, you must go to the Approved tab.

 

STEP 4: To view the denied blood requests, you must go to the Denied tab.

 

STEP 5: To view the archived blood requests, you can go to the Archived tab.

 

STEP 6: To view and approve the Inventory Manager’s request to use the blood bag exclusively for The Red Bank Foundation, you must go to the Partner Request tab.

 

 

How to add and complete bloodletting events?

 

Only the users with a role of Event Manager and System Administrator can add and complete the bloodletting events. The Events here are the Community-Based Mobile Blood Donation (CBMBD) or bloodletting events.

 

 

STEP 1: Go to the Events Page to view all the active events.

 

STEP 2: Click the Add Event button on the upper right corner of your screen to add new events. A dialogue box will appear on the screen with a Add New Event form for you to fill-up.

 

STEP 3: After the bloodletting event, you must complete the event. In the Active Events’ table, look for the Options column and select the Complete Event button of the event that was completed. A dialogue box will appear on the screen with a Complete Event form for you to fill-up completely. It will automatically update the Inventory of your collaborated partner Medical Institution during the event.
